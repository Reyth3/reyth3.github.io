Privacy Policy

This document describes the collecting and use policy of the end-user’s personal data by the LFMSync application. Be sure to read it if you want to know how the data is being processed, stored and used. 

Collecting Data

The application collects the following data: 

  • User’s LastFM credentials;
  • Application telemetry (e.g. crash logs, events etc)

Gathered information

Gathered telemetry is used to improve LFMSync, and last.fm credentials are used to authenticate the user with the service. The data will never be sold, distributed, or transferred to any third party.  

Security

The credentials are not being stored on the user’s device. They are only used for the authentication process through the official API. The only information stored on the end-user’s device is the authentication token, which can be revoked on the last.fm website at any point.
